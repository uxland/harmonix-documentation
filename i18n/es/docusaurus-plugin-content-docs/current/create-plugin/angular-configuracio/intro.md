# Intro

El siguiente documento es una guía de instalación del entorno de desarrollo de un plugin de Harmonix (HxPlugin) en Angular y la inicialización básica de un **HxPlugin**.



Los plugins de Harmonix desarrollados en Angular tienen ciertas limitaciones. Ver la sección `Limitaciones Angular en Harmonix`.



El ejemplo siguiente se explicará usando el Shell del ETC de Primaria como ejemplo, pero es extrapolable a cualquier otro shell. Todo Shell debe ser publicado y consumible por un plugin.
