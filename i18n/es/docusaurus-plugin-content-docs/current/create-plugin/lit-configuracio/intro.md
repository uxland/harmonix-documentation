# Intro

Lit es la librería de renderizado más estándar de la plataforma, y tiene la ventaja de que el Framework Harmonix también está construido en Lit, por lo cual es la tecnología ideal para desarrollar un plugin.



El ejemplo siguiente se explicará usando el Shell del ETC de Primaria como ejemplo, pero es extrapolable a cualquier otro shell. Todo Shell debe ser publicado y consumible por un plugin.
